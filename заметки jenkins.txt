С помощью J можно автоматизировать рутинные операции. Можно рассылать почту, можно создать job и автоматич отвечать на почту. Когда программист пушает билд на git hub - J может подхватить изменения(с помощью плагина), потом код тестируется на качество CI - continuous inspection.

CICICDCD слово continuous синоним,

CI - continuous inspection. Тестируем код на качество.
Statical analysis - когда мы фильтруем базар() не говоря. Т.е. тестирование кода на ошибки не выполняя его.

CI - continuous integration(sharing|сообща). Когда каждый из команды программистов заливает(injecion) только свою часть кода обговоренную c managerom c shared repository. J тут очень сильно помогает. Он создает билд, тестирование билда происходит по 3м направлениям. Unit test, unit integration test, integration test. После этого происходит archive. Integration Framework - договоренность 2х и более людей по 5 вещам. Как мы вводим информацию, как мы создаем новый артефакт, как мы делаем notification, как мы запускаем, как выводим инфу.

CD - continuous delivery. Билд(артифакт|скомпилированый код) должен разворачиваться в какую то среду. Либо это test env, либо staging env. Т.е. это разворачивание в какую - то NON PRODUCTION env.

CD - continuous deployment. Проиходит после окончательного тестирования билда и происходит deployment на PRODUCTION env.

Jenkins - дерижер(режиссер). Т.е. эта пустая коробка и нужно установить плагины. Плагины можно писать, можно использовать готовые.
Job Security - когда вас захотят уволить , чтоб сделать больнее уволить. Мененджер должен бороться с jc.
Test covarage - сколько линий кода покрывает тест.

revision / changes (add, remove, change) - процесс изменения кода.
зачем нужен git? чтобы можно было откатить изменения(roll back).

Чтобы достучаться до вашей системы нужно иметь ваш адрес. (приватный, публичный, хост). Если мы вбиваем что то в адресную строку что не похоже на публичный адрес, то ОС сначало смотрит хост файл. 
Whitespace - (space, tab, new line) что то что отделяет одно от другого. ipaddress [whitespace] nam1 занятие - установить J + редактировать хост файл и поменять порт.
